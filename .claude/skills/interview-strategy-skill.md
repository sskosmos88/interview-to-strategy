# Skill: interview-to-strategy

## Назначение

Этот skill позволяет агенту (Claude) работать с инструментом `interview-to-strategy` — CLI-генератором маркетинговой стратегии из сырой расшифровки стратегического интервью со специалистом.

Инструмент превращает текстовую транскрипцию MP3-интервью в структурированный Markdown-документ, который можно передать специалисту. Документ повторяет структуру `anna-kovaleva-interview-output.md` и включает:

- сводку за 2 минуты;
- аналитику активов и пробелов;
- позиционирование и аудиторию;
- готовые тексты для площадок и контента;
- план первых шагов и метрики.

## Когда использовать

- После проведения стратегического интервью со специалистом.
- Когда есть сырой текст транскрипции, но нет времени вручную собирать стратегию.
- Когда нужно быстро выдать клиенту понятный рабочий документ.

## Артефакты

- **Вход:** `.txt` файл с расшифровкой интервью.
- **Выход:** `.md` файл стратегии, готовый к передаче клиенту.
- **Репозиторий:** `https://github.com/sskosmos/interview-to-strategy` (заменить на реальный URL после публикации).

## Как пользоваться

### 1. Подготовить расшифровку

Поместить текст транскрипции в файл, например:

```text
data/ivanova-interview.txt
```

### 2. Запустить генератор

Без API (локальный fallback):

```bash
python main.py --input data/ivanova-interview.txt --name "Мария Иванова" --output output/ivanova-strategy.md --no-api
```

С Claude API (рекомендуется — качественнее):

```bash
# создать .env из .env.example и прописать ANTHROPIC_API_KEY
python main.py --input data/ivanova-interview.txt --name "Мария Иванова" --output output/ivanova-strategy.md
```

### 3. Проверить выходной файл

Открыть `output/ivanova-strategy.md`, проверить:

- имя специалиста;
- УТП;
- цены и услуги;
- кейсы;
- первые шаги.

Всё, что отмечено **[требует проверки]**, нужно дозаполнить вручную или уточнить у специалиста.

## Структура выходного документа

1. **Сводка за 2 минуты** — УТП, отличие, барьер, три первых действия.
2. **Аналитика активов** — что уже есть у специалиста.
3. **Позиционирование** — аудитория, голос, как показывать рынку.
4. **Готовые тексты** — био, услуги, кейсы, контент-план, скрипт первой беседы.
5. **План продвижения** — первые шаги, метрики, резюме.
6. **Контакты продюсера** — подпись в конце документа.

## Ограничения

- Инструмент не делает транскрибацию MP3. MP3 нужно перевести в текст заранее (Whisper, Яндекс SpeechKit, Sonix и т.п.).
- Качество стратегии зависит от качества транскрипции: чем конкретнее ответы специалиста, тем лучше результат.
- Локальный fallback извлекает только базовые сущности (цены, телефон, опыт). Для полного анализа нужен Claude API.

## Разработка

- Язык: Python 3.10+
- Зависимости: `anthropic`, `python-dotenv`
- Тестовый запуск: `python main.py --input data/sample-transcript.txt --output output/strategy.md --no-api`
